プロセスモジュールの接地
======================================

設計の問題点
************************************************

この事例の装置は大きく分けると以下の部分から構成されています。

- 19インチラック（メインブレーカが設置されている）
- プロセスモジュール
- 搬送系
- 補器類（ポンプ、チラー）

お客様から供給頂いた大元の接地線は初めに19インチラックの電装盤に接続されます。
この電装盤内の端子台で接地線が分岐され、19インチラックの端子台からプロセスモジュールの端子台に
配線、接続されます。

メイン分電盤と19インチラック間の接地線を19インチラックとプロセスモジュールで
共有しているので、この部分は **共通インピーダンス** となっていました。その為に接地線に流れるノイズ電流
が互いの機器に影響を及ぼし通信異常を発生させていました。

メーカの取り扱い説明書を確認すると通信ユニットの接地の目的は機能接地と記載があります。
保護接地に比べ機能接地は電位の変動を極力少なくする様な設計的配慮が必要であったが考慮されていなかった。

対策
************************************************

プロセスモジュールの接地線は19インチラックを経由せず、メイン分電盤から直接配線し
一点接地の形に変更した。

.. image:: ./pm_earthline_improve.png

教訓
************************************************

- ノイズ対策に於いて接地の取り方は非常に重要であると認識する事
- 保護接地と機能接地の目的の違いを理解する事
- ノイズにおける共通インピーダンスの影響を理解する事
- 機器の取扱説明書を熟読し、正しい使い方を良く理解する事
