フィルタ
================================

フィルタとは、下図に示すように、導体を流れる電流のうち、必要な成分を通し、不要な成分を除去する部品や働きを指します。
図ではノイズはグラウンドに流し出していますが、このほかに部品内部でノイズのエネルギーを吸収する場合や、
ノイズ源側に跳ね返す場合（インピーダンスを上げる場合）もあります。

.. image:: ./filter_example_diagram.png

.. image:: ./filter_example_diagram2.png

電子機器のノイズ対策では、通常、ノイズは下図に示すように比較的高周波域に偏在していますので、フィルタには高周波成分を除去する
ローパスフィルタを用います。このローパスフィルタには、インダクタ（コイル）や抵抗、コンデンサなどの汎用部品を使うこともできますが、
ノイズ対策の専用部品であるEMI除去フィルタも使われます。

.. image:: ./lowpass_filter.png

このようなノイズの周波数的な偏在を利用したフィルタの他に、電圧の違いを利用したフィルタ（バリスタなど）、伝導モードの違いを利用した
フィルタ（コモンモードチョークコイルなど）もあります。
また、これらのフィルタ以外にもトランスや光ケーブル、光アイソレータなどを一種のフィルタとして利用する場合もあります。
これらは場合によっては極めて優れたノイズ除去効果を発揮しますが、適用できる場面が限られています。
